Oracle wm_concat 函数说明

更新于 2021-03-07 分类于 数据库 阅读次数： Valine： 本文字数： 1.7k 阅读时长 ≈ 2 分钟

Oracle 数据库的 wm_concat 函数用于将多行数据聚合为单行，从而提供与特定值关联的数据列表，它将以逗号来分割列表。
本文主要介绍 wm_concat 函数的功能以及其在 PostgreSQL 数据库中的替换方法。

PostgreSQL
