PostgreSQL HASH 索引拾遗

更新于 2020-08-23 分类于 数据库 阅读次数： Valine： 本文字数： 2.6k 阅读时长 ≈ 2 分钟

PostgreSQL 的 HASH 索引在 9.6 和 10 版本中有所不同，主要的区别在于 HASH 索引在 9.6 及其之前的版本中不会记录在 WAL 日志中，然而在 10 及其之后的版本这个已经被修复了。
