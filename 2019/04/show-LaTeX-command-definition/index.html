LaTeX 查看命令的定义

更新于 2020-08-23 分类于 排版 阅读次数： Valine： 本文字数： 1.7k 阅读时长 ≈ 2 分钟

最近在使用 LaTeX 撰写文档时需要重定义一个命令的，但是我想要知道该命令的原始定义时怎样的，经过查找发现 LaTeX 在这方面做得还是很不错的。目前，我知道有两种方式可以获取命令的定义：(a) 在文档中使用 \show\command；(b) 在命令行中使用 texdef command。
